Projeto Pessoal

Pokédex

Um catálogo digital de Pokémon que busca dados em tempo real de uma API pública e exibe as informações de forma visual e organizada. Nenhum dado é salvo no próprio site; tudo vem direto da internet ao abrir a página.

O que você pode fazer

Pesquisar qualquer Pokémon pelo nome
Ver tipo, habilidades e estatísticas de cada um
Navegar por uma lista paginada de Pokémon
Design responsivo para celular e desktop
HTML CSS JavaScript PokéAPI Fetch API

Projeto Pessoal

Atlas Hub

Um launcher web que abre um hub com dois apps principais: o FitHub, focado em saúde, treinos e alimentação, e o Atlas, voltado para uso diário com ferramentas como gerador de senhas, metas, diário e utilidades pessoais.

O que você pode fazer

Entrar por um launcher e acessar os apps do hub
Gerenciar treinos, alimentação, progresso e metas fitness
Usar ferramentas diárias como senhas, metas e diário
Contar com autenticação, banco de dados, upload e recursos com IA
HTML CSS JavaScript Node.js Express Supabase Groq SDK Multer Helmet PWA SQL
