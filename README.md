# Portfólio — Matheus de Souza

Portfólio pessoal desenvolvido com HTML, CSS e JavaScript puro. Apresenta minha trajetória, projetos, tecnologias utilizadas e formas de contato.

## Páginas

| Página | Descrição |
|--------|-----------|
| `index.html` | Tela inicial com navegação para as demais seções |
| `sobre.html` | Minha trajetória: de Analista de Suporte a Dev Full Stack |
| `projetos.html` | Projetos desenvolvidos (Pokédex, etc.) |
| `tecnologias.html` | Tecnologias e ferramentas que utilizo |
| `curriculo.html` | Currículo e trajetória profissional |

## Tecnologias

- HTML5
- CSS3 (variáveis, grid, animações)
- JavaScript (Vanilla)
- Google Fonts (Syne + DM Sans)
