Projeto Pessoal

Pokédex

Um catálogo digital de Pokémon que busca dados em tempo real de uma API pública e exibe as informações de forma visual e organizada. Nenhum dado é salvo no próprio site — tudo vem direto da internet ao abrir a página.

O que você pode fazer

Pesquisar qualquer Pokémon pelo nome
Ver tipo, habilidades e estatísticas de cada um
Navegar por uma lista paginada de Pokémon
Design responsivo — funciona no celular e no desktop
HTML CSS JavaScript PokéAPI Fetch API
